# S102 — Jeu 2D en C++

Projet de fin de semestre S102 — IUT d'Orsay, Université Paris-Saclay.
Jeu 2D développé en C++ avec la bibliothèque SDL2.

## Fonctionnalités
- Déplacement du personnage avec gestion des sprites et animations
- Détection de collisions
- Logique d'objets interactifs
- Rendu graphique via SDL2

## Technologies
- C++ — programmation orientée objet
- SDL2 — affichage, gestion des événements et du son
- CodeBlocks — environnement de développement
- Makefile — compilation

## Lancer le projet

### Prérequis
- SDL2 installé sur votre machine
- Compilateur C++ (g++ ou MinGW sur Windows)

### Compilation
```bash
make
```

### Exécution
```bash
./bin/s102
```

## Structure du projet
```
├── src/        # Code source C++
├── include/    # Headers
├── assets/     # Sprites, images, sons
├── SDL2/       # Bibliothèque SDL2
└── bin/        # Exécutable compilé
```
